# Running the code:

## Installation
In freelance-beach-cleanup/ run:
```bash
npm install
```

## Getting Started

Run the development server in freelance-beach-cleanup/:

```bash
npm run start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
